🚀 Quick Start

Add GhostChat to your website in 30 seconds:

Step 1: Add the script tag to your HTML 
<script type="module">
  import GhostChat from "https://cdn.jsdelivr.net/npm/ghost-chat-embed/dist/web.js";

  GhostChat.init({
    apiHost: "https://your-api.com",
    theme: {
      window: {
        title: "AI Assistant",
        darkMode: false
      }
    }
  });
</script>
Step 2: Done! 🎉 Your chat widget is live

⚙️ Advanced Configuration

Customize every aspect of your chat widget:

GhostChat.init({
  // Required: Your AI chat API endpoint
  apiHost: "https://api.yoursite.com",

  // Optional: Theme customization
  theme: {
    accentColor: "#667eea",
    button: {
      size: 56,
      right: 20,
      bottom: 20,
      dragAndDrop: true
    },
    window: {
      title: "Chat Assistant",
      width: 400,
      height: 640,
      darkMode: false
    },
    tooltip: {
      show: true,
      message: "Hi there! 👋"
    }
  },

  // Optional: Localization
  i18n: {
    locale: "en",
    strings: {
      welcome: "Welcome to our chat!",
      placeholder: "Type your message..."
    }
  }
});

🎯 What Makes GhostChat Different?

  • True Shadow DOM Isolation: Unlike other widgets, GhostChat uses Shadow DOM to completely isolate styles. No CSS conflicts, no z-index wars, no layout shifts.
  • Modern Stack: Built with React/Preact, Tailwind CSS, shadcn/ui, and Vercel AI SDK for the best developer and user experience.
  • Production Ready: Battle-tested features like scroll containment, portal-based dropdowns, and smooth animations.
  • Developer Friendly: TypeScript support, comprehensive docs, and easy customization.
  • Open Source: MIT licensed. Contribute, fork, or use it however you want.
