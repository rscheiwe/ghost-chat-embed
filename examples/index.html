Quick Start

To integrate GhostChat into your website, add this script:

<script type="module"> import GhostChat from "https://cdn.jsdelivr.net/npm/ghost-chat-embed/dist/web.js"; GhostChat.init({ apiHost: "http://localhost:3000", theme: { window: { title: "Chat Assistant" }, }, }); </script>
